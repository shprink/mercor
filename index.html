Description

Modal is a Mootools based Class that allows you to interactwith your users.

This Class is highly customizable.

Fork me on GitHub Download

Features

Browser Compatibility Found a bug?

Bootstrap compatible!

Mercor Library supports the fabulous Bootstrap framework.

Beginner No need to know JavaScript!

Define your option in the 'mercor-modal' parameter to create your Modal.

Worried about the way your source code look? No worries the 'mercor-modal' parameter is destroy after use.

<button mercor-modal="{'styles': {'width' : 200,'height' : 200}}" href="#" class="btn">Demo</button>

Intermediate Open a Confirm Box

You might want to ask users permission before to do nasty stuff!

$('modal-confirm').addEvent('click',function(e){
var button = this;
var modal = new MercorModal.Confirm({
	'title': 'Confirm',
	'html': 'Do you want [...] button size?',
	'confirm':{
		'callback': function(){
			button.setStyle('width',300);
		}
	}
});
modal.open();
e.stop();
});

Intermediate Add a button

Adding buttons as never been that easy! 

$('modal-footer').addEvent('click',function(e){
var modal = new MercorModal({
	'styles': {
		'width' : 400,
		'height' : 200
	},
	'title': 'My Modal with footer',
	'html': '<p>My HTML content</p>',
	'buttons':[{ html: 'Close', styles: {}, 
	event: function() { this.close(); }}]
});
modal.open();
e.stop();
});

Intermediate Create a UNIQUE Modal !

Bored of cummun designs? Want to create a unique Modal? It is as easy as the below code. You can easily load an Iframe within a Fullscreen modal with a red Overlay and some buttons.

$('modal-custom').addEvent('click',function(e){
var modal = new MercorModal.Iframe({
	'fullScreen': {
		'active': true
	},
	'iframe':{
		'link':'http://google.com'
	},
	'overlay': {
		'styles': {
			'background': 'red'
		}
	},
	'buttons':[{ html: 'Close', styles: {}, event: function() { this.close(); }},
	           { html: 'Do not click me', styles: {}, event: function() { alert('I told you not to!'); }}]
});
modal.open();
e.stop();
});

Advanced Ajax Requests

Get a Json string from an external source and do whatever you want with it!

Open the data source used for this exemple 
$('modal-request').addEvent('click',function(e){
var button = this;
var modal = new MercorModal.Request({
	'title': 'Request',
	'request':{
		'url': 'demo/data.json',
		'success': function(responseText){
			var  items = JSON.decode(responseText), ol = new Element('ol');
			Array.each(items, function(item, offset) {
				var li = new Element('li',{
					html: '<h1>' + item.title 
					+ '</h1><br/><span>' 
					+ item.desc + '</span>'
				});
				li.inject(ol);
			}.bind(this));
			return ol;
		}
	}
});
modal.open();
e.stop();
});

Alert Provides hints to users! Download

Description

Alert is a Mootools based Class that allows you to display different type of alerts to users.

This Class is highly customizable. Change, the size, the position, the style and even the template of you alerts.

Fork me on GitHub Download

Features

Browser Compatibility Found a bug?

Create a Success message

Success messages are great to let users know about a form successfully sent.

$$('.alertSuccess').addEvent('click',function(e){
	var alert = new MercorAlert;
	alert.open({
		'title': 'Praesent bibendum',
		'text': 'Lorem ipsum dolor sit amet, consectetur elit...',
		'type': 'success' // notice, error, success
	});
	e.stop();
});

Documentation

Method: Constructor var alert = new MercorAlert([options]);

Options

Name type default description
container.id String mercor-alert-container The container id
container.el Object document.body The element Object in which to inject the Alert container.
container.position String top-right Container position. Can be 'top-right', top-left', 'bottom-right', 'bottom-left'
node.id String mercor-alert The node id
node.delay Number 5000 The duration of the node in ms.
node.width Number 300 The node width.
node.opacity Decimal [0,1] 0.9 The node maximum opacity.
node.template String 
<div class="mercor-inner">
<div class="mercor-close" title="Close"></div>
<div class="mercor-title">{TITLE}</div>
<div class="mercor-text">{TEXT}</div>
</div>

Method: open alert.open([options]);

Options

Name type default description
text String (optional) The node Title
title String (optional) The node Text
type String success Define the node style. Can be 'success', 'notice', 'error'

Method: Close alert.close();
